The Salt Virtual Machine is a real-time interpreter for compiled Salt code. Each SCC file is made out of the header and the instructions. Detailed information how such SCC file is built and what instructions are avaible, you should see the doc/scc page. That part is not included in the svm subdirectory, because its used for writing your own Salt compiler, which often has nothing to do with SVM itself.
